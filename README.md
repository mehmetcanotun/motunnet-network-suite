# MotunNet Network Suite

MotunNet is a desktop network management and authorized security auditing suite built with Python and PyQt.

It is designed for internal network visibility, asset management, device discovery, monitoring, and defensive security checks in controlled environments.

## Features

- Network device discovery
- Asset and department management
- Live monitoring
- Port scanning
- VNC integration
- Speed test tools
- Rogue device detection
- ARP spoofing detection
- Export and reporting support
- Defensive security monitoring utilities

## Tech Stack

- Python
- PyQt6
- SQLite / local data storage
- Network utility libraries

## Intended Use

This project is intended for authorized internal network administration, monitoring, and defensive security testing only.

Do not use it to monitor, inspect, or access networks without permission.

## Notes

Some optional monitoring capabilities require additional setup depending on the operating system and capture libraries.
